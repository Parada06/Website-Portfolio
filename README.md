#  Personal Portfolio Website

This is a simple responsive **Personal Portfolio Website** created using **HTML, CSS, and JavaScript**.  
It features a dark/light mode toggle, mobile navigation menu, and sections that showcase personal information, hobbies, and contact details.
## Project Structure
Portfolio/
┣  index.html → Main webpage structure
┣  styles.css → Website styling and theme design
┣  script.js → Handles theme toggle and menu interaction
┗  images/ → Profile and hobby images (e.g., Chess.jpg, Anime.jpg)


---

## Features

✅ **Responsive Design** — Works on desktop and mobile screens  
✅ **Dark/Light Mode Toggle** — Switch between themes easily  
✅ **Interactive Navbar** — Collapses into a menu icon on small screens  
✅ **About & Hobbies Sections** — Introduce yourself and share your interests  
✅ **Contact Form** — Simple message input form  
✅ **Social Icons** — Connect via Facebook, Telegram, or GitHub

---

## Technologies Used
- **HTML5** — Structure and content  
- **CSS3** — Styling, layout, and responsiveness  
- **JavaScript** — Interactivity and theme switching  
- **Font Awesome** — For icons (social media, menu)

---

## How to Use
1. Download or clone this repository.
2. Open the folder in your code editor (like VS Code).
3. Launch the project by opening `index.html` in your browser.
4. Edit the text, images, and links to personalize it.
